# Kimberlite

An experimental game framework

## Instructions

### Build & Run

```bash
  # Initialize build
  $ ./init_build_clang debug

  # Build
  $ ./build_target

  # Run launcher
  $ ./launcher
```

## Dependencies

### Build

- Meson
- Git LFS

### Asset build
- Python 3
- Blender (must be in $PATH)
- Ffmpeg
