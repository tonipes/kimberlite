# Traingrid

Some day this will be a game.

# Build instructions

```

$ build_init.sh   # Initialize build folder
$ build.sh        # Build game
$ assets.sh -b    # Build assets
$ build/traingrid # Run game

```

# Dependencies

- SDL2
- SDL2_mixer
- GLEW
- libzip
- glm

# Build dependencies

- Meson
- Python 3
