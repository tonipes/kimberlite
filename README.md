# Traingrid

Some day this will be a game.

## Instructions

### Build & Run

  $ ./build_init.sh   # Initialize build folder
  $ ./build.sh        # Build game and assets
  $ ./run.sh          # Run game

## Dependencies

### Build

- Meson
- Python 3
  - watchdog
- Git LFS
- Blender (must be in $PATH)
- Doxygen

### Runtime

- SDL2
- SDL2_mixer
- GLEW
- glm
